# Projeto 1 – Jogo dos Dados 

### Descrição

Neste aplicativo foi desenvolvido um jogo de sorteio de dados utilizando números aleatórios. Foram adicionados botões, imagens e componentes de programação para gerar um número entre 1 e 6 sempre que o usuário clicar para jogar. Conforme o número sorteado, a imagem exibida na tela é alterada para mostrar a face correspondente do dado, simulando o lançamento de um dado real de forma interativa.

# Projeto 2 – Jogos sortear Números 

### Descrição

Este aplicativo consiste em um jogo educativo de matemática. Foram utilizados componentes para geração de números aleatórios entre 1 e 1000 e operadores matemáticos para criar contas automaticamente. O usuário deve analisar a operação apresentada e inserir a resposta correta em um campo de texto. O aplicativo verifica a resposta fornecida e pode informar se ela está correta ou incorreta, estimulando o raciocínio lógico e o cálculo mental.

# Projeto 3 – Jogo Gotas

### Descrição: 

Neste jogo foi criado um personagem controlado pelas setas de direção. Foram utilizados sprites, imagens, temporizadores e detecção de colisão para criar a mecânica do jogo. O objetivo é movimentar o personagem pela tela para desviar das gotas que caem do céu. O jogador possui uma quantidade limitada de vidas, que são reduzidas sempre que ocorre uma colisão com uma gota. O jogo também controla a quantidade de gotas presentes na tela, aumentando o desafio e exigindo rapidez e atenção do jogador para sobreviver o maior tempo possível.
