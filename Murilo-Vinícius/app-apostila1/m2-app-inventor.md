# 📚 Bases Tecnológicas Envolvidas

Esta atividade trabalha competências importantes da área de **Informática para Internet**.

### Desenvolvimento de Aplicativos

Os alunos irão aprender:

- Como desenvolver aplicativos utilizando **App Inventor**
- Como trabalhar com **interface gráfica (Design)**
- Como programar utilizando **blocos de programação**

### Documentação de Projetos

Também é importante aprender a **documentar um projeto**, registrando:

- objetivo do aplicativo
- funcionamento
- alterações realizadas
- evidências do desenvolvimento

A documentação é essencial no desenvolvimento de software, pois permite que outras pessoas **entendam, utilizem e mantenham o sistema criado**.
